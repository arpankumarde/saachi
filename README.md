# Saachi AI

Saachi is a companion AI trained on therapeutic-style conversations to support personal growth, task reminders, and emotional wellbeing. It maintains short‑term and long‑term memories so it can remember what you tell it and remind you like a real friend.

Features

- Therapeutic-style conversational support to help overcome insecurities and build better habits
- Persistent short-term and long-term memory for personalized continuity
- Task and reminder management based on spoken or written requests
- Nonjudgmental, supportive responses focused on real-life improvement

Privacy & memory

- Memory is used to provide continuity and relevant reminders. Configure or clear memory according to your preferences.
- Treat Saachi as a supportive tool, not a replacement for professional mental health care.

Quick start

1. Install and run the Saachi service per project setup.
2. Interact through the provided UI or API and ask Saachi to remember tasks or personal details.
3. Manage memory and reminders in the settings.
